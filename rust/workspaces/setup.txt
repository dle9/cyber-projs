# set of packages that share same cargo lock and output directory
- in rust project, mkdir workspace
	- cd workspace

- create Cargo.toml inside of workspace/
	- [workspace]
	members=["bin_crate"]

- cargo new bin_crate
- cargo new lib_crate --lib

- add dependency for lib inside bin
	- addition func in lib_crate/src
	- update bin_crate cargo.toml
		- [dependencies]
		- lib_crate = {path="../lib_crate"}

- use the dependency
	- vim bin_crate/src/main.rs
	- use addition func
